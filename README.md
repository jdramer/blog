## Installation

```sh
brew install hugo
npm i
```

## Development

```sh
npm start
```

To update the JS code go to packages/* and run

```sh
npm start
```

## New article

- should have the extension mmark
- the breadcrumb component should be rerendered `npm run build:sitemap`
